## Development

To run the documentation locally, follow these steps:
1. Clone the repository:
    ```bash
    git clone git@github.com:derp-trade/derp-docs.git
    cd derp-docs
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm run dev
    ```
